# Install & Test

1. Upload theme `torten-bern` and plugin `torten-bern-core` to your WP install.
2. Activate theme and plugin.
3. Install and activate WooCommerce.
4. On plugin activation demo products will be created if no products exist.
5. Install WooCommerce Stripe Gateway plugin for card payments (test keys). For Twint, use the Twint QR button on Thank You page to generate QR and pay manually.

Stripe test card: 4242 4242 4242 4242, any CVC, future expiry.
